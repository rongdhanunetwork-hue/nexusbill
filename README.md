### 🔐 Permission Required

To contribute to this project, you **MUST** be added as a collaborator to the GitHub repository.

1.  **Verify you are logged in** with the correct account:
    ```bash
    git config user.name
    git config user.email
    ```
    *If these do not match the repository owner, you must update them.*

2.  **The owner must add you** as a collaborator on the GitHub repository settings.

3.  **Once added**, you can push normally:
    ```bash
    git push origin main
    ```

**If you cannot push, you are not a collaborator.** Ask the repository owner to add you.
